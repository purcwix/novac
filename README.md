# Novac programming language
an easy to use scripting language desinged for quick general purpose programming and tools,
with it's own package manager shipped built in (nv+)
note: language core syntax and built ins documentation is comming later, for now you'll need to test it out yourself
syntax is js-like, but parnethesis arent needed in statements like until/repeat/if/while......, and will cause bugs if you ever use them in for statements


another note: for using nv+, just write it in the command line to see the usage, and a look at the examples/ directory is helpful if you need syntax examples
